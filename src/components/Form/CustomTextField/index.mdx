# TextField

## About

テキストフィールドは、ユーザーがテキストを入力できるようにするための基本的な入力コンポーネントです。

### カスタムテキストフィールド

今回はカスタムテキストフィールドを作成し、デザインをカスタマイズしています。

カスタム後、主に使うPropsは以下の通りです。

- `label`: ラベル
- `placeholder`: プレースホルダー
- `helperText`: ヘルパーテキスト
- `error`: エラー
- `disabled`: 無効化
- `fullWidth`: 幅100%
- `required`: 必須
- `tooltip`: ツールチップ

特にMuiを業務システムで開発する場合は、細かなUIカスタマイズのニーズが出てくると思います。その際は、[公式ドキュメント](https://mui.com/customization/how-to-customize/)を参照し、細かなアップデートを繰り返し、自社のデザインシステムに近づけていくことをおすすめします。
