# CustomSelect

## 概要

CustomSelectは、Material-UIのSelectコンポーネントをベースにした高機能なドロップダウン選択コンポーネントです。単一選択と複数選択の両方をサポートし、MUIの機能性を活かしながら、よりシンプルで使いやすいUIを提供します。

### 主要な Props

- `label`: ラベル
- `options`: 選択肢の配列（`{ value: string | number; label: string }[]`）
- `placeholder`: プレースホルダー
- `helperText`: ヘルパーテキスト
- `error`: エラー状態
- `disabled`: 無効化
- `required`: 必須
- `tooltip`: ツールチップ
- `size`: サイズ（'small' | 'medium'）
- `fullWidth`: 幅 100%
- `multiple`: 複数選択モード

Tooltip は独自にラベルに組み込んでおり、ユーザーに選択肢の説明を提供します。

その他の props については、[MUI Select API](https://mui.com/api/select/)を参照してください。

## 使用例

### 基本的なSelectフォームの例

<CustomSelect
  label="色を選択"
  options={[
    { value: "red", label: "赤" },
    { value: "blue", label: "青" },
    { value: "green", label: "緑" },
  ]}
  required
/>

