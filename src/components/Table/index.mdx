# CustomTable コンポーネント

`CustomTable`コンポーネントは、データを表示するためのカスタマイズ可能なテーブルです。

## 使用方法

```jsx
import { CustomTable } from "./index";
```

```jsx
<CustomTable />
```

## Props

- `data` : テーブルに表示するデータ
- `columns` : テーブルのカラム情報
