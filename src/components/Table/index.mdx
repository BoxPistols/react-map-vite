# CustomSelect

## 概要

CustomTableは、MUIのTableをベースにカスタマイズされたテーブルコンポーネントです。
MUIの基本能性を活かしながら、より実践的なUIUXを提供するテーブルコンポーネントです。

### 参考テーブル

これは CustomTable の一般的な使用例です

MUIのデフォルトのテーブルをベースに、CustomTableはページネーションの制御、キーワード検索フォルター、表示カラムの選択、CRUD操作のハンドラアイコンなどを提供します。

Exampleとして、何らかのステータスのbool状態を表示するカラムUIも表示させています
