# CustomTable

## 概要

- CustomTable は、MUI の Table をベースにカスタマイズされたテーブルコンポーネントです
- MUI の基本能性を活かしながら、より実践的な UIUX を提供するコンポーネントを目指しています
- ここでは MUI-X は使わず、テーブルの体験デザインとしての基本的なユーザビリティ機能を追加して提供ています
- 必要応じてさらに機能追加や調整をしていく予定です

### 参考テーブル

MUI のデフォルトのテーブルをベースに、CustomTable は以下のオプションが追加されています

- キーワード検索
- 表示カラムの選択表示
- CRUD 操作のためのハンドラアイコン
- ページネーションの数の制御

Example として、何らかのステータスの bool 状態を表示するカラム UI も表示させています
