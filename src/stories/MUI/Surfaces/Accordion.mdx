# Accordion

展開・折りたたみ可能なパネルを表示するコンポーネントです。

## 基本的な使い方

```tsx
import { Accordion, AccordionSummary, AccordionDetails } from '@mui/material'
;<Accordion>
  <AccordionSummary expandIcon={<ExpandMoreIcon />}>
    <Typography>セクション1</Typography>
  </AccordionSummary>
  <AccordionDetails>
    <Typography>セクション1の詳細内容</Typography>
  </AccordionDetails>
</Accordion>
```

## 制御コンポーネント

```tsx
const [expanded, setExpanded] = useState('panel1')

<Accordion expanded={expanded === 'panel1'} onChange={handleChange('panel1')}>
  {/* コンテンツ */}
</Accordion>
```

## 無効化

```tsx
<Accordion disabled>{/* コンテンツ */}</Accordion>
```
