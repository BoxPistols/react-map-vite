# Backdrop

背景をオーバーレイして前面のコンテンツを強調するコンポーネントです。

## 基本的な使い方

```tsx
import { Backdrop, CircularProgress } from '@mui/material'

<Backdrop open={loading}>
  <CircularProgress color="inherit" />
</Backdrop>
```
