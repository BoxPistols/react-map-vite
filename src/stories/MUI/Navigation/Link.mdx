# Link

リンクを表示するコンポーネントです。

## 基本的な使い方

```tsx
import { Link } from '@mui/material'
;<Link href='/'>ホームへ</Link>
```

## underline

```tsx
<Link underline="none">
<Link underline="hover">
<Link underline="always">
```

## カラー

```tsx
<Link color="primary">
<Link color="secondary">
<Link color="inherit">
```

## React Routerとの統合

```tsx
import { Link as RouterLink } from 'react-router-dom'
;<Link component={RouterLink} to='/about'>
  About
</Link>
```
