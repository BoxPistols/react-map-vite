# Inputs（入力コンポーネント）

ユーザーからのデータ入力を受け付けるためのコンポーネント群です。
フォームやインタラクティブな操作に使用します。

## 📝 コンポーネント一覧

### テキスト入力

- **TextField** - 基本的なテキスト入力フィールド

  - variant: outlined（デフォルト）, filled, standard
  - multiline対応で複数行入力も可能
  - helperTextでヒントを表示

- **Autocomplete** - 自動補完付き入力
  - 大量の選択肢から検索して選択
  - 複数選択にも対応
  - カスタムレンダリング可能

### 選択入力

- **Select** - ドロップダウン選択

  - MenuItemで選択肢を定義
  - 複数選択モード対応
  - 選択肢のグループ化も可能

- **Autocomplete** - 検索可能な選択
  - Select の代替として使用可能
  - よりリッチなUI

### Boolean入力

- **Checkbox** - チェックボックス

  - 独立した選択肢
  - indeterminate（不定）状態もサポート
  - FormGroupで複数をグループ化

- **Switch** - トグルスイッチ

  - ON/OFF状態の切り替え
  - 設定画面でよく使用
  - ラベル位置のカスタマイズ可能

- **Radio** - ラジオボタン
  - 排他的な選択肢
  - RadioGroupで必ずグループ化
  - 方向（horizontal/vertical）指定可能

### 数値・範囲入力

- **Slider** - スライダー

  - 範囲内の値を選択
  - 離散値・連続値両対応
  - 範囲選択（min-max）も可能

- **Rating** - 評価入力
  - 星マークなどで評価を入力
  - 小数点精度の設定可能
  - カスタムアイコンに変更可能

### ボタン系

- **ButtonGroup** - ボタングループ

  - 関連するボタンをグループ化
  - variant/size/colorを統一
  - 垂直配置も可能

- **Fab** - フローティングアクションボタン

  - 主要なアクションを強調
  - extended（拡張）モードでテキスト表示
  - 画面の固定位置に配置

- **ToggleButton** - トグルボタン
  - ON/OFF状態を持つボタン
  - 排他的選択・複数選択両対応
  - テキストとアイコンの組み合わせ可能

## 💡 使い分けのヒント

### TextField vs Autocomplete vs Select

| コンポーネント | 選択肢の数 | 検索 | 自由入力 | 用途         |
| -------------- | ---------- | ---- | -------- | ------------ |
| TextField      | -          | -    | ✓        | テキスト入力 |
| Select         | 〜20個     | ✗    | ✗        | 少数の選択肢 |
| Autocomplete   | 20個〜     | ✓    | △        | 多数の選択肢 |

### Checkbox vs Radio vs Switch

| コンポーネント | 選択数  | 状態   | 用途                 |
| -------------- | ------- | ------ | -------------------- |
| Checkbox       | 複数OK  | ON/OFF | リストからの複数選択 |
| Radio          | 1つのみ | 排他的 | グループから1つ選択  |
| Switch         | -       | ON/OFF | 設定のON/OFF切替     |

## ♿ アクセシビリティ

### ラベルの重要性

```tsx
// ❌ 悪い例
<TextField />

// ✅ 良い例
<TextField
  label="メールアドレス"
  helperText="例: user@example.com"
/>
```

### 必須項目の明示

```tsx
<TextField
  required
  label='お名前'
  error={hasError}
  helperText={hasError && 'お名前は必須です'}
/>
```

### キーボード操作

- すべてのコンポーネントはキーボードで操作可能
- Tabキーで移動
- Enter/Spaceで選択

## 🎨 スタイリング

### バリアント

多くのコンポーネントで3つのvariantをサポート:

- **contained** - 背景色あり（目立つ）
- **outlined** - 枠線のみ（中間）
- **text** - テキストのみ（控えめ）

### サイズ

- **small** - コンパクト
- **medium** - 標準（デフォルト）
- **large** - 大きめ

### カラー

- primary, secondary - ブランドカラー
- error - エラー表示
- warning, info, success - 状態表示

## 📋 実装例

### ログインフォーム

```tsx
<TextField
  fullWidth
  label="メールアドレス"
  type="email"
  autoComplete="email"
/>
<TextField
  fullWidth
  label="パスワード"
  type="password"
  autoComplete="current-password"
/>
<FormControlLabel
  control={<Checkbox />}
  label="ログイン状態を保持"
/>
```

### 検索フィールド

```tsx
<Autocomplete
  options={options}
  renderInput={(params) => <TextField {...params} label='検索' />}
  freeSolo
/>
```
